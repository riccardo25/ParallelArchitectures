Description

Main program: after data input, the program must create a link between Mandelbrot set and the output image. This link is the Mandelbrot matrix. This integer matrix contains the number of iterations made by Mandelbrot algorithm to define if a point belongs to the Mandelbrot set (0 means that point belongs to the Mandelbrot set, otherwise > 0 number of iteration made before condition fails). After creation of that matrix the program converts and save the matrix like a BMP file. The location and name of the file are defined by standard values and the user input values. At final because of the stack limitation memory allocation, program must release the dynamically allocated memory for integer Mandelbrot matrix
