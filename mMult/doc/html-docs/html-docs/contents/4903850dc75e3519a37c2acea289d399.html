Mandelbrot Project :: Mandelbrot (simple) :: MatrixCreation

Description

After acquiring the dimensions of the image based on the resolution value, this algorithm allocates the matrix (described in main) in the Heap and for every point of it calculates the equal point of complex plane. After that the algorithm take this complex point and sends it to the Mandelbrot Algorithm to evaluate it. Return 0 in case of no error occurred.
