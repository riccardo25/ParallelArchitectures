Description

The image creation algorithm converts the Mandelbrot matrix to a bitmap image. In order to create and manage the BMP file I used the Montessoro library:

http://web.diegm.uniud.it/pierluca/publichtml/teaching/freesoftware_project.html

The color palette is created randomly. Every iteration number has a specific color. New image has different color palette. File location and name has been defined by standard values or user input.
