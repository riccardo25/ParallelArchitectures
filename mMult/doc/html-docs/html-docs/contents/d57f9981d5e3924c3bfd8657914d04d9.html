Mandelbrot Project :: Mandelbrot (simple) :: main

Description

General description of the main program: this mandelbrot program creates a fractal (scalar) image based on Mandelbrot set. From command line the program receives different types of informations like name of the file that has to be saved, different (from standard value) resolution parameter and so on... All parameters are described in the dedicated area of the documentation.
